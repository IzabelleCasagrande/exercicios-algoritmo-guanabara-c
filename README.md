# exercicios-algoritmo-guanabara-c
Repositório dedicado à resolução dos exercícios de lógica de programação do curso de Algoritmos do prof. Gustavo Guanabara (Curso em Vídeo), transcritos e adaptados para a **Linguagem C**.

## 🎯 Objetivo
Demonstrar a aplicação de lógica de programação, estruturas de controlo e manipulação de dados utilizando uma linguagem de baixo nível estruturada.

## 📂 Estrutura do Repositório
O repositório está organizado por módulos e temas para facilitar a navegação:
- `Modulo01_Sequencias/`: Exercícios de entrada, saída e operações aritméticas básicas.
- `Modulo02_Condicionais/`: Estruturas de decisão (`if`, `else`, `switch`).
- `Modulo03_Repeticoes/`: Estruturas de loop (`while`, `for`, `do while`).
- `Modulo04_Vetores/`: Manipulação de arrays unidimensionais.

## 🛠️ Tecnologias Utilizadas
- **Linguagem C**
- **VS Code** (Ambiente de Desenvolvimento)
- **Git & GitHub** (Controlo de Versão)
